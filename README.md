## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

This site is designed to be deployed on Vercel, Netlify, or any modern web hosting platform that supports Next.js.
